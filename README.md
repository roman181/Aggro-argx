# ARGO Casino – Novo Line Style Slot

Ein kleines, browserbasiertes Demo-Casino mit einem Novo-Line-Style Slot-Automaten
auf Basis von ARGO-HYTOS Bildsymbolen.

## Features

- Casino-Homepage (`index.html`) mit Hero-Bereich und eingebettetem Slot
- Slot-Game (`slot.html`) mit:
  - 5 Walzen, 3 Reihen
  - 1 Gewinnlinie (mittlere Reihe)
  - 6 Bildsymbolen (Logos / Produkte)
  - Einsatz-Einstellung
  - Auto-Play
  - Gewinnberechnung basierend auf einer einfachen Paytable

## Nutzung

1. Repository klonen oder ZIP herunterladen.
2. `index.html` im Browser öffnen.
3. Über die Startseite das Slot-Game spielen.

## Deployment

Für GitHub Pages:

1. Repository auf GitHub erstellen.
2. Dateien hochladen.
3. Unter **Settings → Pages** den Branch `main` und Root auswählen.
4. Warten, bis die Seite gebaut wurde – dann ist das Casino öffentlich erreichbar.

## Hinweis

Dieses Projekt ist eine technische Demo ohne Echtgeld-Funktionalität.
Es dient nur zu Präsentations- und Lernzwecken.
